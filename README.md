# Trading Agent

AI-powered stock investment decision system with multi-agent debate mechanism.

## Overview

Trading Agent is a system that helps individual investors make swing/mid-frequency trading decisions through:

- **News filtering**: Filter relevant news from multiple sources
- **Multi-perspective analysis**: Technical + Fundamental analysis
- **Debate mechanism**: Buy/Sell agents debate, judge decides
- **Portfolio integration**: Combine with existing positions
